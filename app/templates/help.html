About

SchistoProt is a supervised machine learning based tool which uses 17 supervised machine learning techniques using Scikit-Learn to discriminate between i) surface and non-surface proteins and ii) secretory and non-secretory peptides. These 17 techniques are:


1. Machine (GBM)

2. Support Vector Machine with radial basis function (RBF SVM)

3. SVMs with linear kernel (Linear SVM)

4. Decision Tree

5. Random Forest

6. Ada Boost

7. Gaussian Naive Bayes (GNB)

8. Linear Discriminant Analysis (LDA)

9.Quadratic Discriminant Analysis (QDA)

10. Ridge regression

11. Stochastic gradient descent (SGD)

12. Perceptron

13. Passive aggressive

14. Bernoulli Naive Bayes (BNB)

15. Multinomial Naive Bayes (MNB)

16. Nearest Centroid method

17. Multi-layer Perceptron (MLP)



Generated predictions are stored in a database, which facilitates the rapid reuse of predictions without rerunning the time-consuming machine learning classifications. This saves considerabe runtime if the same sequences are uploaded multiple times, e.g. by different users. SchistoProt predictions are highly valuable for identifying genes important for host-parasite interaction and the discovery of novel drug and vaccine targets.

Input options

SchistoTarget takes the fasta formatted text or file. File extension can be any type (eg. .txt, .fas, .fasta etc.). File extension does not have any effect on SchistoTarget.

If a valid email address is provided, an email containing the output result CSV file will be sent automatically. So for large dataset, it is better to provide the email address.

Output

After completing the task, a result page with a prediction table is appeared. The result is also sent by email if a valid email address is provided during data submission.

The prediction table is a list of Sequence ID of test protein sequence, category prediction, decision score and probability.





The feature table is a list of Sequence ID of test protein sequences and the values for 76 features.





Plots and Charts can be drawn from 'Draw Charts' section.
